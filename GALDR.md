# Managed by galdr

This project uses [galdr](https://github.com/wrm3/galdr) for AI-powered task management.

> galdr is a file-based task management and development system that works identically
> across Cursor, Claude Code, Gemini, Codex, and OpenCode — no backend required.

## Quick Reference

| Action | Cursor | Claude Code |
|--------|--------|-------------|
| Initialize | `@g-setup` | `/g-setup` |
| New task | `@g-task-new` | `/g-task-new` |
| Status | `@g-status` | `/g-status` |
| Plan | `@g-plan` | `/g-plan` |
| Review | `@g-review` | `/g-review` |
| Sprint | `@g-sprint` | `/g-sprint` |

## Key Files

- **Tasks**: `.galdr/TASKS.md`
- **Plan**: `.galdr/PLAN.md`
- **Bugs**: `.galdr/BUGS.md`
- **Ideas**: `.galdr/tracking/IDEA_BOARD.md`
