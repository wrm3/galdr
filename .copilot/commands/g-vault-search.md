Search the vault: $ARGUMENTS

## What This Command Does

Searches the file-first vault without requiring MCP or database services.

## Workflow

1. Use `g-skl-vault`
2. Read `index.md` for the human map
3. Read `_index.yaml` for exact routing and metadata
4. Open the most relevant notes
5. Answer from the notes
6. If the result is durable, offer to save it back as a compiled note
