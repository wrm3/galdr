# Pack: startup-tools

VC fundraising, business formation, product development, and resource access for early-stage startups.

## What This Installs

- `startup-vc-fundraising` — Pre-seed through Series C+; pitch decks, investor targeting, term sheets, due diligence
- `startup-business-formation` — Delaware C-Corp, cap table, 83(b) election, founder vesting, foreign qualification
- `startup-product-development` — Discovery, MVP scoping, RICE prioritization, stack choice, QA, build vs buy
- `startup-resource-access` — Cloud credits, AI credits, grants, accelerators, banking tools, community resources

## Prerequisites

None (documentation skill)

## Install

```powershell
.\skill_packs\startup-tools\install.ps1
.\skill_packs\startup-tools\install.ps1 -ProjectRoot "C:\my-project"
```

## Uninstall

Delete these skill directories from your project's `.cursor/skills/`, `.agent/skills/`, etc.:
- `startup-vc-fundraising/`, `startup-business-formation/`, `startup-product-development/`, `startup-resource-access/`
