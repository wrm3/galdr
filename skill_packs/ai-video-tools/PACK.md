# Pack: ai-video-tools

AI video generation, avatar creation, animated GIFs, explainer video production, and multi-platform ad specs.

## What This Installs

- `ai-avatar-lipsync` — OmniHuman, Fabric, PixVerse models; audio-driven lipsync; dubbing workflows
- `ai-video-generation` — 40+ models via inference.sh CLI: text-to-video, image-to-video, Veo, Wan, Seedance
- `animated-gif-creator` — Composable animation primitives, Slack constraints (2MB/64KB), easing + optimization
- `explainer-video` — Script formulas, pacing rules, scene planning, voiceover, music integration
- `storyboard-creation` — Shots, angles, movement, continuity; pre-production for film and animation
- `video-ad-specs` — TikTok/Instagram/YouTube/Facebook/LinkedIn dimensions, timing, AIDA framework
- `remotion-video` — React-based video: compositions, audio, captions, 3D, transitions, cloud render
- `pipeline-validation` — Multi-agent QA gates: specs, narrative, render readiness, asset handoff checks

## Prerequisites

- For `ai-video-generation`: `pip install inference-sh` or API access to inference.sh
- For `remotion-video`: `npm create video@latest`
- For `animated-gif-creator`: ImageMagick or ffmpeg

## Install

```powershell
.\skill_packs\ai-video-tools\install.ps1
.\skill_packs\ai-video-tools\install.ps1 -ProjectRoot "C:\my-project"
```

## Uninstall

Delete the installed skill directories from your project's `.cursor/skills/`, `.agent/skills/`, etc.
