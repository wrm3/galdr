# Pack: blockchain

EVM, Solana, Cosmos, Bitcoin — Solidity, DeFi, NFTs, tokens, cross-chain development.

## What This Installs

- `web3-blockchain` — EVM/Solana/Cosmos/Bitcoin; Solidity smart contracts; DeFi, NFTs, token design; cross-chain bridging

## Prerequisites

- For Ethereum/EVM: `npm install hardhat ethers`
- For Solana: Rust toolchain + `sh -c "$(curl -sSfL https://release.solana.com/stable/install)"`
- MetaMask or wallet for testing

## Install

```powershell
.\skill_packs\blockchain\install.ps1
.\skill_packs\blockchain\install.ps1 -ProjectRoot "C:\my-project"
```

## Uninstall

Delete `web3-blockchain/` from your project's `.cursor/skills/`, `.agent/skills/`, etc.
